Manual de Usuario — TourneyJS

Herramienta educativa para analizar archivos de torneos y generar reportes.

Flujo de uso

  1. Abra index.html en su navegador.
  2. Presione Cargar archivo o pegue el contenido en el área de texto.
  3. Haga clic en Analizar.
  4. Si hay errores léxicos, corríjalos con ayuda de la tabla que indica línea y columna.
  5. Cuando el análisis esté libre de errores, podrá exportar:
    • Tokens (HTML)
    • Errores (HTML)
    • Reportes completos (HTML)
    • DOT del bracket (para Graphviz)

Formato de los archivos

El archivo sigue la estructura de las secciones TORNEO, EQUIPOS y ELIMINACION. Vea el ejemplo en la pantalla principal para una guía.

Visualización de brackets

La aplicación genera el código DOT para Graphviz. Para visualizar el diagrama, use Graphviz local (dot -Tpng bracket.dot -o bracket.png) o un visor online.
